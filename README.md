# 🛍️ Product Listing

A React e-commerce product listing app with cart functionality, built with Vite and DummyJSON API.

🔗 **Live Demo:** https://aillywutw.github.io/product-listing

## Features

- 🔍 Search products by keyword
- 🏷️ Filter by category
- 💰 Sort by price (low to high / high to low)
- 🛒 Shopping cart (add, remove, update quantity)
- 📄 Product detail page
- ✨ Smooth animations (Framer Motion)

## Tech Stack

- React + Vite
- DummyJSON API
- Axios
- React Router DOM
- Framer Motion
- GitHub Pages
